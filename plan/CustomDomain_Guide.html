1
Render 커스텀 도메인 등록
Render 대시보드 설정 경로
  1. Render Dashboard 접속 → kimbap-ui Static Site 선택
  2. 좌측 메뉴 Settings → Custom Domains
  3. Add Custom Domain 클릭
  4. menu.nirvanasoft.kr 입력 → Render가 안내하는 CNAME 값 복사
ℹ️ Render가 안내하는 값은 보통 kimbap-ui.onrender.com 형태의 CNAME 타겟입니다.
2
DNS CNAME 레코드 추가
도메인 관리 업체 DNS 콘솔에서 추가
필드값
TypeCNAME
Name (호스트)menu
Value (타겟)kimbap-ui.onrender.com
TTL300 (또는 Auto)
⚠️ Render가 IP 주소를 안내하는 경우 CNAME 대신 A 레코드로 추가하세요.
🔒 HTTPS 인증서는 Render가 Let's Encrypt로 자동 발급합니다. 별도 작업 불필요.
